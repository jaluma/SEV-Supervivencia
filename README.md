# Juego de Naves

### Prerequisites
```
UnrealEngine 4.16.x
```

### Ampliaciones
 * Interfaz de usuario. Crear un menú principal para entrar al juego y una pantalla de pausa.
 * Minimapa.
 * Balas finitas.
 * Mapa con 4 puertas.
